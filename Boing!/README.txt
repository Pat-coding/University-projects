Flash shapes specification:
(flashed) <r2> <g2> <b2>

- Must include the last 3 integer but can be any value if flashed is false.
- (flashed) is a boolean variable which indicates whether or not the shape will be flashed.
- Example: circle 250 250 -3 1 true 80 255 0 0 false 0 0 0 2000
- It the flash and RGB is inserted between the first colour and insertion time variables.
